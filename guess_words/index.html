Guess Words

Deskripsi Game

Komputer akan memilih kata acak dari daftar kata. Pemain akan diminta untuk menebak huruf dalam kata tersebut. Pemain memiliki sejumlah nyawa (misalnya 6 nyawa). Jika pemain menebak huruf yang salah, mereka kehilangan satu nyawa. Game berlanjut hingga pemain menebak kata dengan benar atau kehabisan nyawa.

Kode Game


import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;
import java.util.Random;
import java.util.Scanner;

public class GuessTheWord {
    public static void main(String[] args) {
        List words = Arrays.asList("java", "programming", "computer", "science", "developer");
        Random random = new Random();
        String wordToGuess = words.get(random.nextInt(words.size()));
        char[] guessedWord = new char[wordToGuess.length()];
        Arrays.fill(guessedWord, '_');

        int lives = 6;
        Scanner scanner = new Scanner(System.in);

        System.out.println("Selamat datang di game Tebak Kata!");
        System.out.println("Kata yang harus ditebak: " + String.valueOf(guessedWord));
        System.out.println("Anda memiliki " + lives + " nyawa.");

        while (lives > 0 && new String(guessedWord).contains("_")) {
            System.out.print("Masukkan huruf: ");
            char guess = scanner.nextLine().charAt(0);

            if (wordToGuess.indexOf(guess) >= 0) {
                for (int i = 0; i < wordToGuess.length(); i++) {
                    if (wordToGuess.charAt(i) == guess) {
                        guessedWord[i] = guess;
                    }
                }
                System.out.println("Bagus! Kata saat ini: " + String.valueOf(guessedWord));
            } else {
                lives--;
                System.out.println("Salah! Anda memiliki " + lives + " nyawa tersisa.");
            }
        }

        if (lives > 0) {
            System.out.println("Selamat! Anda telah menebak kata: " + wordToGuess);
        } else {
            System.out.println("Game over! Kata yang benar adalah: " + wordToGuess);
        }

        scanner.close();
    }
}

Cara Menjalankan Game

  1. Persiapkan Lingkungan: Pastikan kamu memiliki Java Development Kit (JDK) terinstal di komputermu. Kamu bisa mengunduhnya dari situs resmi Oracle.
  2. Buat File Java: Buat file baru dengan nama GuessTheWord.java di editor teks atau IDE favoritmu.
  3. Salin Kode: Salin kode di atas ke dalam file GuessTheWord.java.
  4. Kompilasi Kode: Buka terminal atau command prompt, navigasikan ke direktori tempat kamu menyimpan file, dan jalankan perintah berikut untuk mengompilasi kode: 
    javac GuessTheWord.java
  5. Jalankan Game: Setelah berhasil dikompilasi, jalankan game dengan perintah berikut: 
    java GuessTheWord

Cara Bermain

  • Setelah menjalankan game, ikuti instruksi di konsol untuk memasukkan huruf.
  • Game akan memberi tahu apakah tebakan kamu benar atau salah.
  • Game berlanjut hingga kamu menebak kata yang benar atau kehabisan nyawa.

Penjelasan Kode

  • Import: Mengimpor kelas yang diperlukan untuk menggunakan daftar, acak, dan pemindai.
  • Daftar Kata: Menggunakan daftar kata yang akan ditebak.
  • Variabel: wordToGuess menyimpan kata yang harus ditebak, guessedWord menyimpan huruf yang telah ditebak, dan lives menghitung jumlah nyawa.
  • Loop: Menggunakan loop while untuk meminta tebakan pemain hingga kata yang benar ditebak atau kehabisan nyawa.
  • Kondisi: Memeriksa apakah tebakan pemain benar atau salah dan memperbarui status permainan.
