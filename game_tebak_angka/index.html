Game Tebak Angka

Selamat datang di game Tebak Angka! Coba tebak angka yang dipilih komputer antara 1 dan 100.

Deskripsi Game

Komputer akan memilih angka acak antara 1 dan 100. Pemain akan diminta untuk menebak angka tersebut. Setelah setiap tebakan, komputer akan memberi tahu apakah tebakan pemain terlalu tinggi, terlalu rendah, atau benar. Game berlanjut hingga pemain menebak angka yang benar.

Kode Game


import java.util.Random;
import java.util.Scanner;

public class GuessTheNumber {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Random random = new Random();
        
        int numberToGuess = random.nextInt(100) + 1; // Angka acak antara 1 dan 100
        int numberOfTries = 0;
        boolean hasGuessedCorrectly = false;

        System.out.println("Selamat datang di game Tebak Angka!");
        System.out.println("Saya telah memilih angka antara 1 dan 100.");
        System.out.println("Coba tebak angka tersebut!");

        while (!hasGuessedCorrectly) {
            System.out.print("Masukkan tebakan Anda: ");
            int playerGuess = scanner.nextInt();
            numberOfTries++;

            if (playerGuess < numberToGuess) {
                System.out.println("Tebakan Anda terlalu rendah. Coba lagi!");
            } else if (playerGuess > numberToGuess) {
                System.out.println("Tebakan Anda terlalu tinggi. Coba lagi!");
            } else {
                hasGuessedCorrectly = true;
                System.out.println("Selamat! Anda telah menebak angka " + numberToGuess + " dengan " + numberOfTries + " percobaan.");
            }
        }

        scanner.close();
    }
}

Cara Menjalankan Game

  1. Persiapkan Lingkungan: Pastikan kamu memiliki Java Development Kit (JDK) terinstal di komputermu. Kamu bisa mengunduhnya dari situs resmi Oracle.
  2. Buat File Java: Buat file baru dengan nama GuessTheNumber.java di editor teks atau IDE favoritmu.
  3. Salin Kode: Salin kode di atas ke dalam file GuessTheNumber.java.
  4. Kompilasi Kode: Buka terminal atau command prompt, navigasikan ke direktori tempat kamu menyimpan file, dan jalankan perintah berikut untuk mengompilasi kode: 
    javac GuessTheNumber.java
  5. Jalankan Game: Setelah berhasil dikompilasi, jalankan game dengan perintah berikut: 
    java GuessTheNumber

Cara Menjalankan Game dengan Online Compiler

  1. Buka browser dan kunjungi situs JDoodle atau OnlineGDB.
  2. Pilih bahasa pemrograman Java.
  3. Salin kode di atas dan tempelkan ke dalam editor kode di situs tersebut.
  4. Klik tombol "Run" untuk menjalankan kode.
  5. Ikuti instruksi di konsol untuk memasukkan tebakan.

Cara Bermain

  1. Setelah menjalankan game, ikuti instruksi di konsol untuk memasukkan tebakan.
  2. Game akan memberi tahu apakah tebakan kamu terlalu tinggi, terlalu rendah, atau benar.
  3. Game berlanjut hingga kamu menebak angka yang benar.
